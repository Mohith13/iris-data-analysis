# iris-data-analysis


# Iris Statistical Analysis Assignment

This repository contains a statistical analysis of the Iris dataset.

Tasks completed:
- Descriptive statistics by species
- Central tendency and dispersion
- Visualization (histogram, boxplot)
- Skewness analysis
- Central Limit Theorem sampling
- Correlation analysis
- Hypothesis testing (t-test and ANOVA)

All code, plots, and outputs are included.
